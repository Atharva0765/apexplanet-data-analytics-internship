# ApexPlanet Value-Based Customer Segmentation

## Task 3: Deep-Dive Analysis & Interactive Dashboarding

### Overview
Comprehensive customer segmentation analysis identifying High-Value, Medium-Value, and Low-Value customer segments with strategic insights and interactive Power BI dashboard.

### 📊 Key Metrics
- **Total Customers:** 947
- **Total Revenue:** ₹139.4 Crore
- **High-Value Segment:** 313 customers, ₹91.8 Cr (66% of revenue)
- **Growth Opportunity:** ₹25-35 Crore

### 🎯 Key Insights
1. **Revenue Concentration:** 33% of customers drive 66% of revenue
2. **Upsell Opportunity:** Medium-Value customers only ₹58K away from High-Value
3. **Strong Loyalty:** All segments show >100% repeat purchase rate
4. **Category Preference:** Electronics/Education drive High-Value spending
